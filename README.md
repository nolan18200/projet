## 🚀 Lancer le projet

### Sur Android (Acode)
1. Ouvre `index.html` dans **Acode**
2. Lance le serveur via le bouton **Live Server**
3. Accède au site via `http://localhost:8158/index.html`

### Sur PC
1. Ouvre le dossier dans **VS Code**
2. Lance **Live Server** (clic droit → Open with Live Server)
3. Accède au site via `http://127.0.0.1:5500/index.html`

## 🎨 Stack technique

- **HTML5** — Structure des pages
- **CSS3** — Styles, animations, responsive
- **JavaScript** — Interactions, animations au scroll
- **PHP** — Traitement du formulaire de contact
- **JSON** — Données du portfolio
- **Google Fonts** — Police Inter

---

## ✨ Fonctionnalités

- Design **dark mode** avec dégradé violet/rose/orange
- Animations au scroll via **IntersectionObserver**
- 100% **responsive** mobile
- Redirection automatique sur la page 404 après 10 secondes
- Navigation fluide entre toutes les pages
- Respect du **RGPD**
